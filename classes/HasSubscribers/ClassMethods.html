auto_subscribes(subscriber, options = {}) Link

Subscribe an associated user to an associated object when this record is created. For example, you might auto-subscribe a comment user to the blog post they commented on UNLESS they authored the blog post:

auto_subscribes :user, :to => :blog_post, :if => {|comment, blog_post| comment.user_id != blog_post.user_id}

Options:

  • :to - association to call to retrieve the user

  • :if - block called to determine whether or not to create the subscription. Takes the record and the subscribable as args.

notifies_subscribers_of(subscribable_association, options = {}) Link

Tell a model to generate updates for subscribers of an association. For example, a Comment notifies_subscribers_of :blog_post.

Configuration options:

  • :on - event that triggers notification. Values: update or create.

  • :with - name of an instance method that notifies subscribers. Use this if you want to customize the way updates get generated.

  • :if - block to decide whether to send generate an update. Takes the following arguments: notifier, associate, subscription. So if a comment was going to generate updates for subscribers to its parent blog post, the arguments would be comment, blog_post, subscription, and this block would get called for every subscription.

  • :queue_if - block to decide whether to queue a record for update generation. The :if block determines whether the record generates an update, but that still happens in a Delayed::Job. :queue_if determines whether that job gets delayed in the first place. Takes the record as its arg.

  • :priority - DJ priority at which to run the notification

  • :include_owner - Create an update for the user associated with the resource, e.g. if a comment generates an update for an observation, the owner of the observation should be notified even if they're not subscribed. This can also be a Proc that takes the arguments subscribable and notifier.

  • :include_notifier - Create an update for the person associated with the notifying record.

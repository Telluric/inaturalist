get_lineage_for(taxon) Link

Finds a taxon's ancestors from the name provider and returns an array of them as new Taxon objects up until there is one already in our database. Thus, the first Taxon in the array should either be a new Kingdom or an existing saved Taxon that is already in our local tree.

   # File lib/ratatosk/lib/ratatosk/name_providers/catalogue_of_life.rb
45 def get_lineage_for(taxon)
46   # If taxon was already fetched with classification data, use that
47   hxml = if taxon.class != Taxon && taxon.hxml && taxon.hxml.at('classification')
48     taxon.hxml
49   elsif !taxon.source_identifier.blank?
50     @service.search(:id => taxon.source_identifier, :response => 'full' )
51   end
52   lineage = [taxon]
53 
54   if hxml
55     # walk UP the CoL lineage creating new taxa
56     [hxml.search('classification/taxon')].flatten.reverse_each do |ancestor_hxml|
57       lineage << ColTaxonAdapter.new(ancestor_hxml)
58     end
59   end
60   lineage.compact
61 end

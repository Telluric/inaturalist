get_lineage_for(taxon) Link

Finds a taxon's ancestors from the name provider and returns an array of them as new Taxon objects up until there is one already in our database. Thus, the first Taxon in the array should either be a new Kingdom or an existing saved Taxon that is already in our local tree.

Source: show

38 def get_lineage_for(taxon)
39   # If taxon was already fetched with classification data, use that
40   #TODO use historically cached info
41   if taxon.class != Taxon && taxon.hxml && taxon.hxml.at('ClassificationHierarchy')
42     hxml = taxon.hxml
43   else
44     hxml = @service.get_taxon_for_nzor_id(:nzor_id => taxon.source_identifier)
45   end
46   lineage = [taxon] #don't put the taxon in there to begin with as it is a part of the classification hierarchy
47 
48   # walk UP the NZOR lineage creating new taxa
49   hxml.at('ClassificationHierarchy').children.reverse_each do |ancestor_hxml|
50     temp_taxon = NZORTaxonAdapter.new(ancestor_hxml)
51     lineage << temp_taxon unless taxon.source_identifier == temp_taxon.source_identifier
52   end
53   lineage.compact
54 end

validates?(operand) Link

validates? takes 'thing' and runs a test on it by calling the operator via send. validates? passes the dynamically called operator, 'thing' along with anything defined in parameters. (parameters are stored in the database as a pipe deliminated string).

This requires that rules, which may defined as any class level method, gracefully handle the objects they receive.

Utilitity for benchmarking code in the console. Use it like this:

Benchmarker.benchmark do |b|

b.benchmark "thing 1" do
  # do thing 1
end
b.benchmark "thing 2" do
  # do thing 1
end

end

Every time you call `b.benchmark “thing 1”` the time of the block will be recorded and stats about how long “thing 1” calls took will be reported at the end
